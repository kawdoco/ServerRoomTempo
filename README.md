<h1><center><strong>Server Room Management System </center></h1>
<h2>Aims of the Project </h2>
<p><li> Ensure secure and efficient management of server room operations.</li></p>
<li>Monitor environmental conditions (temperature, humidity, power usage). </li>
<li> Generate alerts and reports for proactive issue resolution. </li>

  <h2>Scope </h2>
  <h3><strong> The system will include:</strong> </h3>
  <li>Dashboard Interface: Real-time monitoring of server room conditions.</li>
  <li>Alert System: Notifications for abnormal conditions (e.g., overheating).</li>
  <li>Reporting Tools: Generate usage, maintenance, and incident reports.</li>

  <h2>Methodology </h2>
  <li>Requirement Gathering: Interview IT staff and facility managers.</li>
  <li>System Design: Define architecture (web-based or hybrid)</li>
  <li>Implementation: Develop modules for monitoring, access, and reporting.</li>
  
  <h2>Resources </h2>
  <li>ESP32 </li>
  <li>DHT22 (temperature + humidity, digital output, good accuracy)</li>
  <li>Micro‑USB cable and adapter (5V, ≥1A)</li>
  <li>Breadboard & Jumper Wires  </li>
  <li>Resistors  
(Typically 4.7kΩ pull‑up resistor for DS18B20, or as required by sensor datasheet)</li>
<h2>📡 Communication & Networking</h2>
<li>Wi-Fi Router/Access Point  
(ESP32 will connect to this for sending data)</li>

<h2>🔔 Outputs & Monitoring</h2>
<li>Sending Alerts Via Messages </li>
<li>Create Logs for future incidents </li>
<li>Utlise Logs to establish Incident Recovery Strategy  </li>
<li>Use this logs as an inputs to  Comprehensive System Monitoring </li>

<h2>Reaseach and Installation Location </h2>
<li>BCI Campus Negombo - Sri Lanka </li>
